Bio

Hi, I'm Soroush Maleki (abysseraphim). I'm a self-taught security practitioner focused on web application security, offensive security, and security research. I enjoy understanding how applications work by reading code, testing assumptions, and digging one layer deeper than the obvious.

My approach is manual-first: I prefer understanding a system before automating it. Rather than relying on automated scanners or blindly firing payloads, I start with harmless requests, observe how an application behaves, and build an understanding of the system before moving to more advanced testing. In my experience, understanding how an application behaves often reveals issues that automated scanning or blind payloads can easily miss.

Along the way, I build open-source tools to support my research, document what I learn through write-ups, and continuously explore new techniques in application security.
