Lurak

Lurak is a lightweight, modular, and Linux-first reconnaissance framework designed for web application security assessments and penetration testing. It solves the pain of manually chaining dozens of tools by orchestrating them into a clean, repeatable, and transparent workflow — while keeping every module independent and all outputs reusable. From ASN intelligence and subdomain enumeration to service discovery and smart pipelines, Lurak lets pentesters focus on analysis instead of glue work.

TaintHound

TaintHound parses JavaScript source code (from a local project or a live URL) into an AST, identifies known taint sources (user-controllable input) and dangerous sinks (execution/DOM/network primitives), and traces data-flow paths between them to surface potential client-side vulnerabilities such as DOM-based XSS, open redirects, and unsafe eval/Function usage.

Paramancer

A concurrent HTTP parameter mutation engine for web application security testing. Paramancer probes every parameter in a request — name and value — using systematic mutations, then compares each response against a clean baseline to surface anomalies. No scanner. No plugin. Just structured noise with a purpose.

FreeCollapse

Freecollapse takes an input string and generates mutations by inserting byte values at every position — or specific ones you choose. The idea comes from a real problem in web security testing: parsers, WAFs, and sanitizers often behave differently depending on where a special byte appears. Freecollapse makes it easy to generate all those variations automatically and pipe them into your testing workflow. The name and concept are inspired by reCollapse — a great tool focused on URL normalization fuzzing. Freecollapse takes the same core idea but extends it: not just URLs, not just fixed positions, but arbitrary input with full control over byte ranges and insertion indices.
